Python Programming Basics

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Key Features:
- Easy to learn and understand
- Dynamic typing
- Automatic memory management
- Extensive standard library
- Interpreted execution


Data Types in Python:
1. Integers: Whole numbers like 5, -10, 100
2. Floats: Decimal numbers like 3.14, -2.5
3. Strings: Text data enclosed in quotes
4. Booleans: True or False values
5. Lists: Ordered collection of items
6. Dictionaries: Key-value pairs
7. Tuples: Immutable sequences
8. Sets: Unordered collections of unique items

Variables and Assignment:
Variables are containers for storing data values. In Python, you don't need to declare the data type; it is automatically determined when you assign a value.

Example:
name = "John"
age = 25
height = 5.9
is_student = True

Control Flow:
Python uses if-elif-else statements for conditional execution and loops for repetitive tasks.

Functions:
Functions are reusable blocks of code that perform specific tasks. They are defined using the def keyword and can accept parameters and return values.
